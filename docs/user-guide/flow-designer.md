# 决策流设计器

基于 bpmn-js 的可视化决策流设计器，用于编排多个规则节点的执行顺序。

## 相关 API

- `GET /{root}/ruleflowdesigner/loadFlowDefinition` — 加载决策流定义
- `POST /{root}/packageeditor/loadFlows` — 列出项目中的决策流

## 决策流结构

决策流 JSON 包含：

- **nodes** — 节点列表（规则节点、开始/结束节点、网关等）
- **connections** — 节点间的连线
- **ruleRefs** — 规则引用（关联到具体的规则集、决策表等）

## 流程控制

基于 Flowable 8 BPM 引擎，支持：

- **串行执行** — 节点按顺序依次执行
- **并行执行** — 并行网关同时执行多个分支
- **条件分支** — 排他网关根据条件选择分支
- **包容分支** — 包容网关根据条件执行一个或多个分支

## 节点类型

决策流中的节点可引用以下规则类型：

- 规则集（向导式/脚本式）
- 决策表
- 决策树
- 评分卡
- 脚本节点
